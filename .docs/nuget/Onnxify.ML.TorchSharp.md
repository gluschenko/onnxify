# Onnxify.ML.TorchSharp

`Onnxify.ML.TorchSharp` is a publishable TorchSharp-oriented ML package slot in this repository.

## Install

```bash
dotnet add package Onnxify.ML.TorchSharp
```

## Intended Direction

- TorchSharp-oriented ML helpers that complement `Onnxify.ML`.
- Higher-level model workflows that build on the repo's ONNX and TorchSharp foundations.
