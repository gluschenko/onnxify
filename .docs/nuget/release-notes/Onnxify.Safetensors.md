## 0.1.2

- Added a generic `SafeTensors` value API with `Set<T>(...)`, `Get<T>(...)`, and `Remove(...)` so callers can store supported CLR arrays without manually marshaling bytes into `TensorView`.
- Added async file helpers with `LoadFromFileAsync(...)`, `SerializeToFileAsync(...)`, and `SaveToFileAsync(...)`.
- Added round-trip coverage for all supported generic value element types.
