## Description

GRUTraining operator is adapted from GRU operator (https://github.com/onnx/onnx/blob/main/docs/Changelog.md#GRU-14).The difference between the two operators is that GRUTraining generates an additional output:the intermediate zrh gate outputs.This extra output is needed for the gradient computation while training.
