## Description

FakeQuant operator that fuses quantization->dequantization pattern into a single node. FakeQuant takes in a non quantized tensor as input and generates a non quantized tensor as output. But internally, it will perform Quantization->Dequantization operation that simulates the effects of quantization within the model. Loss in numerical precision introduced by model quantization is corrected by adjusting the model weights through the FakeQuant op.
